# Angular

### 1. nodejs 설치 후 버전확인
``` 
npm -v
```
<br>

### 2. typescript설치
```
npm install -g typescript
```
<br>

### 3. angular CLI 설치
```
npm install -g @angular/cli
```
<br>

### 4. 프로젝트 생성
```
ng new 프로젝트명
```
